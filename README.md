rfcdaemon
=========

An easier way to manage the reading of Request For Comments (RFCs) at www.rfc-editor.org.

Usage
=====
options:

--object/-o		--action/-a		--more

index			info			-n <name>

index			list			-n <name>

index			create			-n <name>

index			add				-n <number>

rfc				info			-n <number>

rfc				obsolete		-n <number>

rfc				get				-n <number>

rfc				list
